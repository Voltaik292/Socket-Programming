ENCS3320 - File Request System

Request Networking Files

How it works:

  • Enter the name of the image or video file you want (e.g., "NetworkFirewall.png" or "NetworkAttacks.mp4")
  • Select whether you're searching for an image or video
  • If the file exists on our server, it will be displayed.
  • If the file doesn't exist, you'll be redirected to Google search results for similar images or videos
  • Image requests redirect to Google Images search
  • Video requests redirect to Google Videos search
